# dotfiles

My personal dotfiles.

- OS: NixOS
- WM: Hyprland
- Shell: Bash
- Editor: NeoVim
- Theme: Catppuccin
- Browser: Zen-Browser

Additionally, there are QMK configuration files for the Cheapino keyboard.

## Installation

Setup is managed via a Makefile. To view available commands and usage details, run:
```sh
make help
```
